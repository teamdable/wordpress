# Dable for WordPress

워드프레스용 데이블 플러그인입니다. 

## 설치

워드프레스 플러그인 폴더에 `dable-for-wordpress` 폴더를 만들고 그 안에 PHP 코드를 저장하면 됩니다.

## 기능

* 데이블 전용 메타 태그 생성.
* Open Graph 메타 태그 생성.
* 컨텐트를 `<div itemprop="articleBody">`로 래핑.
